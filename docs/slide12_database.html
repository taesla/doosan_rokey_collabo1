데이터베이스 선정 (Database Selection)

Why Firebase?
DB 솔루션 비교
Database 실시간 설치 비용 ROS2 통합 호스팅
Firebase RTDB WebSocket 즉시 무료* Python SDK 자체
PostgreSQL 폴링 설치 필요 무료 psycopg2 별도
MongoDB Change Stream 설치 필요 Atlas 무료 pymongo Atlas
Redis Pub/Sub 설치 필요 무료 redis-py 별도
AWS DynamoDB Streams 즉시 유료 boto3 자체
SQLite 없음 내장 무료 sqlite3 로컬

* Firebase Spark Plan: 1GB 저장, 10GB/월 전송 무료

Firebase 선정 이유
  • 실시간 동기화: WebSocket 기반으로 데이터 변경 즉시 UI 반영
  • Zero Configuration: 서버 설치/관리 불필요, 즉시 사용
  • 상태 복구: 웹 종료 후 재시작 시 마지막 상태 자동 복원
  • 무료 티어: 소규모 프로젝트에 충분한 무료 할당량
  • Python SDK: firebase_admin으로 ROS2 노드에서 직접 연동
