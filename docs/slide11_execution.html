시스템 실행 방법 및 결론

Run & Wrap-up

Option 1: 전체 시스템 실행 (Launch)

$ cd ~/cobot1_ws
$ source install/setup.bash
$ ros2 launch dsr_integrated full_system.launch.py

Option 2: 개별 노드 실행

$ ros2 run arduino_conveyor_bridge serial_to_topic
$ ros2 run dsr_integrated sort_node
$ ros2 run dsr_integrated web_server_node

Web Access

Browser URL: http://<Robot_PC_IP>:5000
